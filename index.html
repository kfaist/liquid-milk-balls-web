About

Interactive web-based art project with WebRTC streaming capabilities for TouchDesigner integration via OBS and NDI.

üé• LiveKit Streaming Setup

For production-quality streaming with OBS, NDI, and TouchDesigner integration:

Complete Bidirectional Workflow:

  1. Publisher Page: Remote user captures webcam and publishes to LiveKit
  2. NDI Viewer Page: View the LiveKit stream (capture this in OBS Browser Source)
  3. OBS: Capture NDI viewer page and enable NDI output (Tools ‚Üí NDI Output Settings)
  4. TouchDesigner: Use NDI In TOP to receive stream from OBS, process & stylize
  5. TD Output: Use NDI Out TOP to send processed video back to OBS
  6. OBS Capture: Add NDI Source in OBS to receive processed video from TD
  7. Return Viewer Page: Remote user views processed video with fullscreen option

UI Guide

Mirror Section:

  • Mirror Circle: Click anywhere on the mirror to create ripple effects and change the echo message
  • Create Echo: Generates multiple ripples at once and changes the mirror's color every 5 clicks
  • Clear: Resets the mirror to its initial state and message

WebRTC Section:

  • Local Camera: Video preview of your webcam feed (displays after clicking Start Camera)
  • Remote Stream: Video preview of the peer's webcam feed (displays after WebRTC connection)
  • Start Camera: Requests access to your webcam and displays it in Local Camera preview
  • Stop Camera: Stops the camera feed and releases the webcam (enabled after camera starts)
  • Start WebRTC Call: Connects to the signaling server and establishes peer-to-peer connection (enabled after camera starts)
  • Hang Up: Ends the WebRTC call and closes the peer connection (enabled during active call)
  • Status Display: Shows real-time status messages about camera and connection state
